# LLM Agentic Book Bot

An AI-powered multi-agent system that autonomously generates book chapters using LLMs.

## Features
- Web content scraping
- Agent-based planning and writing system
- Automated chapter generation
- Modular architecture

## Tech Stack
Python, Large Language Models, Agent Design, Web Scraping

## How it Works
1. Scraper collects source content  
2. LLM agents analyze and plan structure  
3. Writing agent generates chapter  
4. Output saved to final_chapter.txt
